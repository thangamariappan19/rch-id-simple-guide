### Why This App?

Many families face confusion and anxiety when visiting Government Hospitals for the first time. This app aims to:

- âœ… **Reduce Anxiety** by explaining the hospital process step-by-step
- âœ… **Prevent Repeat Visits** by clarifying document requirements upfront

## ğŸ“¦ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/rch-id-simple-guide.git
   cd rch-id-simple-guide
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run locally**
   ```bash
   npm start
   ```
   Navigate to `http://localhost:4200/`

### Build for production
   ```bash
   npm run build
   ```
   Output will be in `dist/rch-id-simple-guide/browser` directory.
